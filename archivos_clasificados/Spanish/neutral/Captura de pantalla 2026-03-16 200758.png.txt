Enunciado. Ejercicio Red Neuronal con regularización
Dropout
Dado un conjunto de puntos generado aleatoriamente en 4 clases, realizad una red neuronal para la correcta
clasificación.
Podéis poner las capas intermedias que consideréis
.
Una vez realizada, incluid una regularización Dropout en cada capa intermedia de tasa 0.2
Comparad los datos obtenidos en ambas redes.
Nota: Cómo crear un conjunto de datos aleatorios de 4 clases:
from sklearn.datasets import make_classification
X, y = make_classification(n_features=2, n_classes=4, n_samples=1000, n_redundant=0,
n_informative=2, random_state=47, n_clusters_per_class=1, scale=0.25)
Se puede imprimir el conjunto de datos para verlo visualmente con colores diferentes:
plt.scatter(X[ :, 0], X[ :, 1], c=y.astype(np. float))
